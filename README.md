# Personal Portfolio Website

This is my personal portfolio website created using **HTML**, **CSS**, and **JavaScript**.  
The website includes my skills, projects, education details, certifications, and a downloadable resume.

##  Features
- Clean and responsive UI  
- Home, About, Skills, Projects, Education & Certifications sections  
- Downloadable resume  
- Smooth design and easy navigation  
- Hosted using **GitHub Pages**

##  Technologies Used
- **HTML5** – For structure  
- **CSS3** – For styling  
- **JavaScript** – For interactivity  

##  Folder Structure
portfolio/
│── index.html
│── style.css
│── script.js
│── README.md


##  How to Run the Project
1. Download or clone the repository  
2. Open **index.html** in any browser  
3. The website will run locally without any server
